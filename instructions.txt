kvault CLI - Setup Guide

Install dependencies:
   npm install

Link the CLI globally:
   npm link
This will allow you to use the 'kvault' command anywhere on your system. Be in kvault/vault-cli

Initialize a new vault:
   kvault init
   This creates a .vault folder in your current directory with a config file.

Authenticate the CLI:
   First, log in on the kvault web app and download your CLI token.
   Then run:
   kvault import-token path/to/token.json

   This stores your refresh token and user info in .vault/config.json.
